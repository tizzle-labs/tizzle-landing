02

Protocol Description

Tizzle is a decentralized event ticketing protocol built on the Solana blockchain. The Platform enables:

  • Stake-to-Attend: Users stake tokens into a smart contract escrow to register for events, replacing traditional ticket purchases
  • On-Chain Attendance Verification: Attendance is verified via QR code scanning at the event venue, recorded immutably on-chain
  • Refund Mechanism: Staked tokens are returned to attendees after verified attendance, subject to organizer configuration and unlock time
  • Pay-per-Slot Model: Event organizers pay a flat platform fee per attendee slot; attendees are never charged a platform fee

All core protocol logic is executed by open smart contracts (programs) on the Solana blockchain. Token custody is handled exclusively through Program Derived Addresses (PDAs) — Tizzle does not hold or control user funds directly.
