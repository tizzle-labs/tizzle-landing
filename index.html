FAQ

Common questions.

Do I need a crypto wallet to join the waitlist?

No. The waitlist is just an email. You only need a wallet when the app launches.

What is Solana devnet?

Devnet is Solana's test network — real blockchain infrastructure with no real money involved. Perfect for a hackathon demo.

How does ticket verification work?

Each ticket is a signed on-chain token. The venue scanner reads the QR, checks the signature against the chain, and gets an instant yes/no.

Can tickets be transferred or resold?

Yes — because tickets live in your wallet, you can transfer them like any token. Resale rules are set by the organizer at mint time.

Is this production-ready?

Not yet. This is a hackathon demo on devnet. We're validating the concept before moving to mainnet.

What wallets are supported?

We're targeting Solana mobile wallets (Phantom, Solflare). Any wallet that supports the Solana token standard will work.
